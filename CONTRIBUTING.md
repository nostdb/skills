# Contributing to nostos-skills

The Skill sources are available for evaluation, but external contributions are closed pending legal/CLA approval. Do not submit workflows, fixtures, patches, or pull requests expecting inclusion; [CLA.md](CLA.md) is status only.

Use [SECURITY.md](SECURITY.md) for vulnerabilities. Prospective contributors can run [README verification](README.md#verify). When intake opens, both adapters must retain identical canonical Skill bytes and all deterministic graph behavior must continue through the public CLI/Core boundary.
